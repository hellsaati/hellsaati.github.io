# Həll saatı 10. 2D massivlər

Cədvəl, matris, oyun taxtası, labirint kimi halların hamısı iki ölçülü massivlərdir. 2D massiv isə "massivlərin massividir": `a[i][j]` i-ci sətir, j-ci sütun.

## Declare etmək, oxumaq, çap etmək

```cpp
const int MAXN = 1005;
int a[MAXN][MAXN];

int n, m;
cin >> n >> m;

for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        cin >> a[i][j];

for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++)
        cout << a[i][j] << ' ';
    cout << '\n';
}
```

2D massiv yaddaşda sətir-sətir düzülür. `a[i][j]`-də birinci indeks sətir, ikinci sütundur.

## Sətir və sütun əməliyyatları

```cpp
// i-ci sətrin cəmi
long long row = 0;
for (int j = 0; j < m; j++) row += a[i][j];

// j-ci sütunun cəmi
long long col = 0;
for (int i = 0; i < n; i++) col += a[i][j];
```


## Diaqonallar

n * n kvadrat matrisdə:

```cpp
// Əsas diaqonal: (0,0), (1,1), ..., (n-1,n-1)
for (int i = 0; i < n; i++) cem += a[i][i];

// Yan diaqonal: (0,n-1), (1,n-2), ..., (n-1,0)
for (int i = 0; i < n; i++) cem += a[i][n - 1 - i];
```

Əsas diaqonal üzərində `i == j`, yan diaqonal üzərində isə `i + j == n - 1` şərti ödənir.

## Qonşu xanalar

```cpp
int di[] = {-1, 1, 0, 0};
int dj[] = {0, 0, -1, 1};

for (int k = 0; k < 4; k++) {
    int ni = i + di[k];
    int nj = j + dj[k];
    if (ni < 0 or ni >= n or nj < 0 or nj >= m) continue;
    // (ni, nj) mövcud qonşudur
}
```

:::danger[Diqqət]
Sərhədlərə diqqət edin, massivdən kənara çıxmayın. Əlavə olaraq diaqonallar olarsa, 8 qonşu üçün di/dj massivlərini 8 elementli yazın.
:::

## Yerdəyişmə və döndərmə

```cpp
// a[i][j] indeksləməsini a[j][i] -yə keçirmək(matrisi 90 dərəcə çevirmək)
for (int i = 0; i < n; i++)
    for (int j = i + 1; j < n; j++)
        swap(a[i][j], a[j][i]);
```

`j = i + 1`-dən başlamaq vacibdir: bütün cütləri gəzsək, hər cüt iki dəfə dəyişilib geri qayıdar.

## Nümunə: Magic Square

n * n matris "sehrli" matrisdirmi? Sehrli elə matrisə deyilir ki, həmin matrisdə bütün sətir, sütun və iki diaqonalın cəmləri bərabərdir.

```cpp
#include <iostream>
using namespace std;

int a[105][105];

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            cin >> a[i][j];

    long long g = 0;
    // 0-cı sətrin cəmi
    for (int j = 0; j < n; j++) g += a[0][j];

    bool ok = 1;

    // sətirlər
    for (int i = 0; i < n; i++) {
        long long s = 0;
        for (int j = 0; j < n; j++) s += a[i][j];
        if (s != g) ok = 0;
    }

    // sütunlar
    for (int j = 0; j < n; j++) {
        long long s = 0;
        for (int i = 0; i < n; i++) s += a[i][j];
        if (s != g) ok = 0;
    }

    // diaqonallar
    long long d1 = 0, d2 = 0;
    for (int i = 0; i < n; i++) {
        d1 += a[i][i];
        d2 += a[i][n - 1 - i];
    }
    if (d1 != g or d2 != g) ok = 0;

    cout << (ok ? "YES" : "NO");
    return 0;
}
```

## Çalışmalar

**Çalışma 1.** Matrisdə ən böyük elementin sətir və sütun nömrəsini tapın.

<details>
<summary>Həlli</summary>

```cpp
int bi = 0, bj = 0;
for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        if (a[i][j] > a[bi][bj]) { bi = i; bj = j; }
cout << bi + 1 << ' ' << bj + 1;
```

</details>

**Çalışma 2.** Matrisin sərhəd elementlərinin (birinci/sonuncu sətir və sütun) cəmini hər elementi bir dəfə saymaqla tapın.

<details>
<summary>Həlli</summary>

```cpp
long long cem = 0;
for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        if (i == 0 or i == n - 1 or j == 0 or j == m - 1)
            cem += a[i][j];
```


</details>

**Çalışma 3.** n * n matrisi spiral üzrə (saat əqrəbi ilə, bayırdan içəri) çap edin.

<details>
<summary>Həlli</summary>

```cpp
int up = 0, dw = n - 1, le = 0, ri = n - 1;
while (up <= dw and le <= ri) {
    for (int j = le; j <= ri; j++) cout << a[up][j] << ' ';
    up++;
    for (int i = up; i <= dw; i++) cout << a[i][ri] << ' ';
    ri--;
    if (up <= dw)
        for (int j = ri; j >= le; j--) cout << a[dw][j] << ' ';
    dw--;
    if (le <= ri)
        for (int i = dw; i >= up; i--) cout << a[i][le] << ' ';
    le++;
}
```
Ortadakı `if`-lər tək sətir/sütun qalanda təkrar çapın qarşısını alır.

</details>

## Tez-tez edilən səhvlər

1. Sətir/sütun indekslərini qarışdırmaq (`a[j][i]`).
2. Qonşularla iş zamanı sərhədləri yoxlamamaq.
3. Böyük 2D massivi lokal elan etmək (stack overflow).
4. Yerədiyşmədə bütün cütləri gəzib matrisi geri qaytarmaq.
5. Çapda sətir sonu `'\n'`-in yerini səhv salmaq.
