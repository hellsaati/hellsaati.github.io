# Giriş: Olimpiada mühitinə giriş, datatiplər və dəyişənlər

Salam, mən Əli Süleymanzadə. Respublika İnformatika Olimpiadalarına son 10-cu sinifin ortalarında başlamağıma baxmayaraq 11-ci sinifdə gümüş medal qazandım. Bununla yanaşı, bir sıra respublika(RM 90, Iostream CUP) medal qazanmaq və beynəlxalq(APIO 2025 və APIO 2026) olimpiadalarda iştirak etmək şansım oldu. Bu yolda çox çətinliklərim oldu, zaman-zaman gecələrlə yuxusuz qalıb həm məktəb tapşırıqlarımı, həm də olimpiada hazırlığımı davam etdirməli oldum. Lakin, çəkdiyim əziyyətin qarşılığını aldım və ADA Universitetində pulsuz təhsil almaq haqqı qazandım.

## İnformatika Olimpiadalarına necə hazırlaşmaq olar?
İnformatika Olimpiadalarına hazırlaşmaq əslində subyektiv bir prosesdir: Ola bilər ki, siz yeni prinsipləri digərlərindən daha tez və ya daha yavaş öyrənirsiz. Bu öyrənmə prosesi əslində çox önəmli bir faktordur: Hər mövzunu o qədər yaxşı mənimsəməlisiniz ki, o mövzuya ehtiyac duyduqda müəyyən açar nöqtələr anında beyninizdə canlana bilsin. Bir kod parçasını yazanda eyni yerdə dəfələrlə buraxdığınız səhv açar nöqtələrə misal ola bilər.

Proqramlaşdırmanın müxtəlif növləri var, amma bizim öyrənəcəyimiz növə ümumi olaraq *competitive programming*, yəni yarış proqramlaşdırması deyilir.

İnformatika Olimpiadalarında ən çox istifadə olunan iki dil var: **C++** və **Python**. Lakin, daha əlverişli və sürətli olduğu üçün yarışlarda C++ dilinin istifadəsi daha məqsədəuyğundur.

## Bəs CP(Competitive Programming) üzrə necə daha yaxşı ola bilərəm?

Bu cavab, yəqin ki, sizə tanışdır: daha çox sual işləməklə. Həqiqətən, olimpiada mühiti elə qurulub ki, oturub sadəcə alqoritmləri öyrənərək heç bir şeyə nail olmaq mümkün deyil.

Sual işlədikdə də sizə asan və ya çox çətin gələn sualları deyil, sizə azca çətin gələn sualları seçməlisiniz ki, artıq o suallar sizin üçün asanlaşsın və siz səviyyəni biraz da artırasınız.

Bir mövzunu yeni öyrəndikdə, yaxşı olar ki, sırf o mövzuya aid ən azı 5-10 sual işləyəsiniz. Tam mənimsədikdən sonra isə başqa suallarla qarışdırıb işləmək daha yaxşıdır, çünki artıq o sual tipini ayırd etməyi öyrənməlisiniz.

Sırf bir mövzu haqqında öyrənmək üçün və ya suallar tapmaq üçün müxtəlif resurslar, bloglar tapa bilərsiniz, [məsələn](https://codeforces.com/blog/entry/57282).

## İlk kod

Compiler və Code Editor-ları yüklədiyimizə görə, artıq ilk kodumuzu yazmağa keçə bilərik. Yeni bir fayl yaradıb, adını istədiyiniz formada qoyun. Lakin, faylın tipinin C++ faylı olduğundan, yəni `.cpp` ilə bitdiyindən əmin olun.

İndi isə həmin faylın içində aşağıdakı kodu yazın:

``` cpp
#include <iostream>
using namespace std;

int main(){
    cout << "Hello, World!" << endl;
    return 0;
}
```

***Qeyd***: Proqramlaşdırmada yeni dil və ya Code Editor öyrənərkən ilk yazılan proqram adətən "Hello, World!" olur. Bu proqramın məqsədi sadəcə hər şeyin düzgün qurulduğunu yoxlamaqdır. Kodu icra etmək üçün onu "compile" etmək, yəni maşın dilinə çevirmək lazımdır.

***Qeyd***: Kodu yazdıqdan sonra kodu "compile" etməmişdən qabaq ***CTRL + S*** basaraq kodu yadda saxlayın.

## Əsas hissələrin izahı

Gəlin sətir-sətir kodu analiz edək.

C++ ilə birlikdə gələn bir çox hazır kitabxanalar mövcuddur. Biz bu kitabxanalardan istifadə edərək ekrana yazı yaza, istifadəçidən məlumat ala və bir çox başqa əməliyyatları yerinə yetirə bilərik. `#include <iostream>` də ***iostream*** adlı kitabxananın koda daxil edildiyini göstərir. Kitabxanalar haqqında gələn dərs daha ətraflı danışacayıq.

`using namespace std` sətri haqqında çox öyrənməyə ehtiyac yoxdur, sadəcə kodumuzda bu funksiyaları daha qısa şəkildə yazmağımız üçün istifadə edirik.

`int main` əsas funksiyanı göstərir. Proqram işə düşdükdən sonra icra `main()` funksiyasından başlayır. Funksiyalar haqqında gələn dərs daha ətraflı öyrənəcəyik. `{}` mötərizələri, qısacası funksiyanın içini göstərir.

`cout` bizə hər hansısa məlumatı ekrana çap etməyə kömək edir. Kodumuzda olmasa da, `cin` funksiyası da mövcuddur ki, dəyişənlər bölməsində nümunələrinə baxacayıq. 

`"Hello, World!"` bir string məlumat növüdür. Məlumat növləri(datatiplər) və dəyişənlər haqqında bundan sonra danışacağıq. Qısacası, string çoxlu simvolların(char) birləşməsi kimi düşünmək olar.

`<<` işarəsi isə `cout`-a aiddir. Bu işarələr sayəsində kompüter hansı məlumatı ekrana çapa verməli olduğunu anlayır.

`endl` isə sadəcə `Enter` kimi düşünmək olar. Kompüterin kursorunu yeni sətrə keçirir.

`return 0` isə funksiyanın 0 qaytardığını göstərir. `return` sözündən sonra funksiya dayanır. Deməli, `return 0` əslində əməliyyat sisteminə 0 qiymətini qaytarır, hansı ki kodun normal işləyib bitdiyini göstərir.


## ";" işarəsi

Sətirlərin çoxunun axırında ";" işarəsi var. Bu nə üçündür?

Bunu cümlələrin sonundakı "." işarəsi kimi düşünə bilərsiniz. Məsələn biz yuxarıdakı `main()` funksiyasında kompüterə "`Hello, World!` yazın. 0 qaytar." deyirik. Hər əmrin sonunda yazılan ";" işarəsi də kompüterə həmin əmrin bitdiyini bildirir.

## Dəyişənlər

İndiyə qədər yazdığımız proqram sadəcə ekrana əvvəlcədən yazdığımız mətni çap edirdi. Bəs əgər proqramın hansısa məlumatı yadda saxlamasını istəsək necə?

Məsələn:

- Bir şagirdin yaşı
- Bir yarış iştirakçısının xalı
- Bir məhsulun qiyməti
- Bir insanın adı

Bu kimi məlumatları proqramın yadda saxlaması üçün dəyişənlərdən (variables) istifadə edirik.

Məsələn:

``` cpp
int age = 13;
```

Burada age adlı bir dəyişən yaratdıq və onun içində 13 qiymətini saxladıq.

Artıq proqramın istənilən yerində 13 yazmaq əvəzinə sadəcə age yaza bilərik.

``` cpp
cout << age;
```

Çıxış:
``` javascript
13
```

Dəyişənlərin adı təsadüfi seçilməyib.

Onların qiymətini proqram işlədiyi müddətdə dəyişmək mümkündür.

``` cpp
int xal = 10;
xal = 25;
cout << xal;
```

İlk olaraq `xal` dəyişəninin qiyməti 10 idi.

Sonra ona yeni qiymət verdik və artıq dəyişən 25 qiymətini saxlayır.

Çıxış:
``` javascript
25
```

## Datatiplər

Bəs *int* nədir?

*int* əslində datatipdir. Müxtəlif datatiplər var: *int*, *char*, *double*, *long long*, *string*, və s. Gəlin hər birinin nəyi ifadə etdiyinə baxaq.

### int

Tam ədədlər üçün istifadə olunur. Təxminən **−2*10⁹** ilə **2*10⁹** arasındakı ədədləri saxlaya bilir.

```cpp
int yas = 13;
int xal = 250;
int il = 2026;
```

### long long

Çox böyük tam ədədlər üçün istifadə olunur. Təxminən **−2*10¹⁸** ilə **2*10¹⁸** arasındakı ədədləri saxlaya bilir.

İnformatika olimpiadalarında sadəcə çox böyük ədədlərlə işləyərkən adətən *long long* istifadə olunur.

```cpp
long long say = 8100000000;
long long cavab = 1000000000000;
```
***Qeyd***: Daha çox məlumat saxladığı üçün, *long long* tipi *int* tipinə nəzərən ikiqat daha çox yaddaş istifadə edir.

### double

Kəsr hissəsi olan ədədlər üçün istifadə olunur.

``` cpp
double pi = 3.14159;
double qiymət = 14.75;
```

### char

Yalnız bir simvol saxlamaq üçün istifadə olunur.

``` cpp
char simvol = 'x';
char reqem = '5';
char sual = '?';
```

***Qeyd***: Diqqət edin ki, *char* dəyişənlərində simvollar tək dırnaq (') arasında yazılır.
***Qeyd***: Diqqət edin ki, *char* dəyişənlərində istisna olan bir simvol var: **"\\"**. Bu istisna ondan ibarətdir ki, *char* olaraq bunu yazmaq istədikdə `'\\'` kimi yazmalıyıq.

### string

Mətn saxlamaq üçün istifadə olunur.

``` cpp
string country = "Azerbaijan";
string food = "Dolma";
```

*string* məlumatları isə qoşa dırnaq (") arasında yazılır.
