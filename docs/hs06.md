# Həll saatı 5. Massiv üzərində əməliyyatlar

Massivi oxumağı öyrəndik; indi onun üzərində hesablamalar aparaq. Bu dərs CP-nin "vurma cədvəli" kimidir. Çünki bu dərs keçəcəklərimiz demək olar ki, hər məsələnin alt məsələsi kimi qarşınıza çıxa bilər.

Bu dərsdə öyrənəcəyik:

- cəm, orta, min/max, ikinci maksimum tapılması;
- qonşu elementlərin müqayisəsi;
- ardıcıl parçalar və onlarla əlaqəli məsələlər.

## Cəm və orta qiymət

```cpp
long long cem = 0;
for (int i = 0; i < n; i++) cem += a[i];

double orta = (double)cem / n;
```

İki incəlik:

1. Cəm üçün `long long` götürmək: 10⁵ ədəd * 10⁹ = 10¹⁴, `int`-ə sığmır.
2. `(double)cem / n`: əvvəl tipi çeviririk, yoxsa `cem / n` tam bölmə olar və kəsr hissə itər.

## Min, max və onların yerləri

```cpp
int minYer = 0, maksYer = 0;
for (int i = 1; i < n; i++) {
    if (a[i] < a[minYer]) minYer = i;
    if (a[i] > a[maksYer]) maksYer = i;
}
```

Bir gedişdə hər ikisini tapırıq. Qiymət lazımdırsa: `a[minYer]`, `a[maksYer]`.

## İkinci maksimum

Maksimumu tapib, silib, yenidən maksimumu tapmaq ideyası yanlış ideyadır. Aşağıdakı ideya daha effektivdir:

```cpp
int m1 = -2000000000, m2 = -2000000000;  // m1 >= m2
for (int i = 0; i < n; i++) {
    if (a[i] > m1) {
        m2 = m1;      // köhnə maksimum ikinci olur
        m1 = a[i];
    } else if (a[i] > m2) {
        m2 = a[i];
    }
}
```

Bu nümunədə sıra kritikdir: əvvəl `m2 = m1`, sonra `m1 = a[i]`. Tərsini yazsanız hər ikisi eyni qiymət olar. Belə "dəyişənlərin estafeti" texnikası CP-də çox işlənir.

:::danger[Bərabər elementlər]
Məsələdən asılı olaraq "ikinci maksimum" ya "ikinci ən böyük qiymət" (fərqli olmalı), ya da "ikinci yer" (bərabər ola bilər) deməkdir. Birinci halda `else if (a[i] > m2 and a[i] != m1)` yazın. Məsələ şərtini diqqətlə oxuyun!
:::

## Qonşu elementlər

Bir çox məsələ qonşu elementlərin müqayisəsinə gəlir: "massiv artan ardıcıllıqla düzülüb?" və ya "ən uzun artan ardıcıllıqla düzülmüş parça".

```cpp
// Massiv artan ardıcıllıqla düzülüb?
bool artan = true;
for (int i = 1; i < n; i++)
    if (a[i - 1] > a[i]) artan = false;
```

Dövr `i = 1`-dən başlayır və `a[i]`-ni `a[i-1]` ilə müqayisə edirik, buna görə də `a[-1]` sərhəd xətasından qaçına bilirik.

### Ən uzun artan ardıcıllıqla düzülmüş parça (Kefa and First Steps)

```cpp
int le = 1, cv = 1;
for (int i = 1; i < n; i++) {
    if (a[i] >= a[i - 1]) le++;
    else le = 1;
    if (le > cv) cv = le;
}
```

İdeya: Cari artan ardıcıllıqla düzülmüş parçanın uzunluğunu artırmağa çalışrıq, sıra pozulanda 1-ə sıfırlayırıq. Bir gedişə edirik deyə zaman səmərəliyi O(n) olur.

## Massivin dəyişdirilməsi

```cpp
// Hər elementi 2 dəfə artır
for (int i = 0; i < n; i++) a[i] *= 2;

// İki elementin yerini dəyiş
int tmp = a[i];
a[i] = a[j];
a[j] = tmp;
// və ya sadəcə: swap(a[i], a[j]);
```

`swap` C++-ın hazır funksiyasıdır və çox işlədəcəyik (xüsusilə, sıralamalarda).

## Nümunə: IQ test

Codeforces 25A: n ədəddən biri cütlüyünə görə qalanlardan fərqlidir; onun nömrəsini tapın.

Plan: cütlərin və təklərin sayını sayaq; azlıqda olanın yerini çap edək.

```cpp
#include <iostream>
using namespace std;

int a[105];

int main() {
    int n;
    cin >> n;
    int ct = 0;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        if (a[i] % 2 == 0) ct++;
    }

    int cs = (ct == 1) ? 0 : 1;  // 0: yeganə sayda cüt var, 1: yeganə sayda tək var
    for (int i = 0; i < n; i++) {
        if (a[i] % 2 == cs) {
            cout << i + 1;
            break;
        }
    }
    return 0;
}
```

İki gediş: birincidə sayırıq, ikincidə tapırıq. Zaman səmərəliyi O(2 * n) = O(n) olur.

## Çalışmalar

**Çalışma 1.** Massivdə maksimum ilə minimum arasındakı fərqi tapın.

<details>
<summary>Həlli</summary>

```cpp
int mn = a[0], mx = a[0];
for (int i = 1; i < n; i++) {
    if (a[i] < mn) mn = a[i];
    if (a[i] > mx) mx = a[i];
}
cout << mx - mn;
```

</details>

**Çalışma 2.** Massivdə "zirvə" elementlərin sayını tapın. "Zirvə" elementləri elə elementlərdir ki, hər iki qonşusundan ciddi böyükdürlər(kənar elementlər zirvə sayılmır).

<details>
<summary>Həlli</summary>

```cpp
int say = 0;
for (int i = 1; i < n - 1; i++)
    if (a[i] > a[i - 1] and a[i] > a[i + 1]) say++;
cout << say;
```

Sərhədlərə baxın: `i` 1-dən n−2-yə qədər gedir, ona görə `a[i-1]` və `a[i+1]` həmişə mövcuddur.

</details>

**Çalışma 3.** Massivi yerində (əlavə massivsiz) tərsinə çevirin.

<details>
<summary>Həlli</summary>

```cpp
for (int i = 0; i < n / 2; i++)
    swap(a[i], a[n - 1 - i]);
```

`i`-ci element `n−1−i`-ci ilə yer dəyişir. Dövrü yalnız yarıya qədər aparırıq, yoxsa hər cüt iki dəfə dəyişilib geri qayıdar.

</details>

## Tez-tez edilən səhvlər

1. Cəmdə `int` daşması (overflow);
2. Tam bölmə ilə ortanı hesablamaq (`cem / n` əvəzinə `(double)cem / n`);
3. İkinci maksimumda mənimsətmə sırasını qarışdırmaq;
4. Qonşuların müqayisəsində `i = 0`-dan başlayıb `a[-1]`-ə girmək;
5. Tərsinə çevirmədə dövrü n-ə qədər aparıb massivi əvvəlki halına qaytarmaq.
