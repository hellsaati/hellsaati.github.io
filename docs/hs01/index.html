Compiler, Code Editorlar və onların yüklənmə qaydası. Kodu compile etmək.

Compileri yazdığınız kod parçasını maşın dilinə - kompüterin anlaya biləcəyi dilə çevirən tərcüməçi kimi düşünə bilərsiniz.

Code Editor-u isə sizin kod yazmağınız üçün dizayn olunmuş proqramdır. Bu proqramlar adətən kodun compile və debug(analiz) edilməsi üçün nəzərdə tutulmuş alətlərlə əvvəlcədən təchiz olunur.

Ən çox istifadə edilən kod editorlara Visual Studio Code, Code::Blocks, Jetbrains IDE, Sublime Text və s. misal göstərmək olar. Hər birinin üstünlükləri olsa da, sadəcə Visual Studio Code və Code::Blocks haqqında öyrənmək bəs edir.

VS Code​

VS Code təkbaşına C++ kodunu işlədə bilmir. Bunun üçün ona compiler də lazımdır. Hansı əməliyyat sistemini işlətməyinizdən asılı olmayaraq, terminalı(command prompt) açıb g++ --version yazanda xəta vermirsə, deməli artıq sizin kompüterinizdə compiler yüklənib, bir daha yükləməyə ehtiyac yoxdur.

Windows​

VS Code-u yükləyin və açın.

Yükləmə prosesində aşağıdakı seçimləri aktiv edin:

  • Add "Open with Code" action to Windows Explorer (File)
  • Add "Open with Code" action to Windows Explorer (Directory)
  • Register Code as an editor for supported file types
  • Add to PATH(requires shell restart)

Finish basın və yükləməni bitirin.

Compilerin yüklənmə qaydası​

MSYS2 yükləyin və açın.

Açılan pəncərədə hərşeyi olduğu kimi buraxıb davam edin, sadəcə ən sonda Run MSYS2 now seçimini aktiv edin.

Yeni pəncərə açılacaq. Bu pəncərəyə pacman -S --needed base-devel mingw-w64-ucrt-x86_64-toolchain yazın.

Aşağıdakı səhifə gəldikdə sadəcə Enter düyməsinə basın.

Son olaraq, bu compileri Windows-un PATH qovluğuna əlavə etməlisiniz ki, Windows hər yerdə onu görə bilsin. Bunun üçün axtarış yerinə Edit environmental ... yazın və Edit environment variables for your account-u seçin. User variables hissəsindən PATH-i tapın və Edit seçin. Açılan hissədə New seçin və C:\msys64\ucrt64\bin yazib OK basın. Əgər VS Code açıq qalıbsa, bağlayıb yenidən açın.

Linux​

Ubuntu/Debian​

VS Code-u yükləyin.

Yuklədiyin qovluğun içində terminal açın və bunu yazın: sudo apt install ./code*.deb

Fedora/RHEL​

VS Code-u yükləyin.

Arch​

Terminalı açın və bunu yazın: sudo pacman -S code.

Compilerin yüklənmə qaydası​

Linux əməliyyat sistemlərində əksər hallarda compiler hazır yüklənmiş olaraq gəlir, bunu g++ --version yazaraq yoxlaya bilərsiniz. Əks halda discord kanalında soruşa bilərsiniz.

Mac OS​

Mac-lər haqqında çox məlumatım olmasada, aşağıdakı tutorial əksər hallarda işə yaramalıdır.

VS Code-u yükləyin və açın.

Visual Studio Code.app faylını götür və Applications qovluğuna atın.

Compilerin yüklənmə qaydası​

Mac OS sistemlərində əksər hallarda compiler hazır yüklənmiş olaraq gəlir, bunu g++ --version yazaraq yoxlaya bilərsiniz. Əks halda terminala xcode-select --install yazaraq yükləyə bilərsiniz.

Əlavə​

Əməliyyat sistemindən asılı olmayaraq, Intellisense-i aktiv etməyi tövsiyyə edirəm. Bu sizə kod yazan zaman yazdığın sözü tamamlamağa kömək edən bir alətdir.

VS Code-da Ctrl + Shift + X basın və Axtarış yerinə "C++" yazın.

Birinci çıxan extensionu yükləyin.

Code::Blocks​

Linux və Mac OS işlədirsizsə, Code::Blocks yükləməyini tövsiyyə etmirəm. Onun yerinə Visual Studio Code, Clion və s. işlədə bilərsiniz.

Windows​

Buradan Microsoft Windows(64 bit, default) hissəsini tapın və sonu ...-mingw-setup.exe ilə bitən faylı yükləyin.

Hərşeyi defolt seçimlərdə saxlayaraq yükləməni bitirin.

Kodu compile etmək​

Visual Studio Code​

Müxtəlif yollar olsa da, mən birdəfəlik terminaldan "compile" etməyi tövsiyyə edirəm. Ctrl + J basıb terminalı açın. Əgər fayl adını main.cpp-dən fərqli birşey qoymusunuzsa, aşağıdakı əmri uyğun olaraq düzəldin.

Əməliyyat sisteminizə uyğun olaraq, aşağıdakılardan birini yazın:

Windows: g++ main.cpp -o main && .\main.exe Linux/Mac OS: g++ main.cpp -o main && ./main

və Enter-ə basın.

Bunu ilk dəfə yazmaq kifayətdir. Sonrakı dəfə terminalda ↑ (yuxarı ox) düyməsini basaraq əvvəlki əmri gətirib sadəcə Enter basa bilərsiniz.

Code::Blocks​

Sadəcə F9 basmaq kifayətdir.
