# Həll saatı 14. STL: vector

Buradan etibarən C++-ın əsl gücünə — **STL**-ə (Standard Template Library) keçirik. STL — hazır, sürətli, sınanmış data strukturlar və alqoritmlər kolleksiyasıdır. İlk və ən vacib üzv: `vector` — **ölçüsü dəyişə bilən massiv**.

## Niyə vector?

Adi massivin iki problemi: ölçü sabitdir və ölçünü ayrıca dəyişəndə daşımalıyıq. `vector` hər ikisini həll edir:

```cpp
#include <vector>

vector<int> v;              // bosh
vector<int> v(n);           // n eded 0
vector<int> v(n, 7);        // n eded 7
vector<int> v = {3, 1, 4};  // hazir qiymetlerle

v.push_back(5);             // sona elave et — O(1)
v.pop_back();               // sondan sil — O(1)
v.size();                   // element sayi
v.empty();                  // boshdurmu?
v[i];                       // indeksleme — massiv kimi
v.back();                   // sonuncu element
v.clear();                  // hamısını sil
```

İnput template-i artıq belədir:

```cpp
int n;
cin >> n;
vector<int> a(n);
for (int i = 0; i < n; i++) cin >> a[i];
```

Ölçü əvvəlcədən bilinmirsə:

```cpp
vector<int> a;
int x;
while (cin >> x) a.push_back(x);
```

:::tip[Massiv yoxsa vector?]
CP-də hər ikisi işlənir; biz bundan sonra **vector-a üstünlük verəcəyik**: ölçü çevikliyi, funksiyalara `&` ilə təmiz ötürmə, STL alqoritmləri ilə uyğunluq. Qlobal böyük massiv yalnız 2D cədvəllər və DP masaları üçün qalacaq.
:::

## Gəzmə üsulları

```cpp
for (int i = 0; i < (int)a.size(); i++) cout << a[i];  // indekslə

for (int x : a) cout << x;          // qiymetle
for (int &x : a) x *= 2;            // reference ilə
```

`size()` unsigned qaytarır.

## Daxil etmə və silmə

```cpp
a.insert(a.begin() + pos, x);   // pos mövqeyinə x — O(n)
a.erase(a.begin() + pos);       // pos mövqeyini sil — O(n)
```

`begin()` — vectorun əvvəlinə "göstərici" (iterator).

O(n) olduqlarını unutmayın: dövr içində `insert/erase` çağırmaq O(n²)-dir. Çoxlu ortadan silmə lazımdırsa, 6-cı dərsdəki "two pointers ilə compress" trick-i vector-da da işləyir.

## 2D vector

```cpp
int n, m;
cin >> n >> m;
vector<vector<int>> a(n, vector<int>(m));   // n x m cedvel

for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        cin >> a[i][j];
```

Üstünlüyü: ölçü icra zamanı seçilir, funksiyaya rahat ötürülür. Sətirlərin uzunluğu fərqli də ola bilər — bu, qraf dərslərində "qonşuluq siyahısı" kimi əvəzsiz olacaq:

```cpp
vector<vector<int>> qonsu(n);
qonsu[2].push_back(5);    // 2-nin qonshusu 5
```

## Faydalı hazır funksiyalar (ilk tanışlıq)

```cpp
#include <algorithm>

sort(a.begin(), a.end());              // siralama — O(n log n)
reverse(a.begin(), a.end());           // tersine cevir
int mx = *max_element(a.begin(), a.end());
int mn = *min_element(a.begin(), a.end());

#include <numeric>
long long cem = accumulate(a.begin(), a.end(), 0LL);   // 0LL — long long!
```

`accumulate`-dəki `0LL`-ə diqqət: `0` yazsanız cəm `int`-də hesablanır və daşır. 

`sort`-un dərin imkanları (comparator, pair) — 16-cı dərsin mövzusudur.

## Nümunə: Missing Number (CSES 1083)

1..n ədədlərindən biri çatışmır; tapın.

**Yol 1 — cəm düsturu:**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long long n;
    cin >> n;
    long long cem = n * (n + 1) / 2;
    for (int i = 0; i < n - 1; i++) {
        long long x;
        cin >> x;
        cem -= x;
    }
    cout << cem;
    return 0;
}
```

**Yol 2 — işarə massivi:** `vector<bool> var(n + 1)`, oxunanları işarələ, işarəsizi tap.

Hər ikisi O(n). Birincisi yaddaşsız, ikincisi ümumiləşə biləndir ("hansı **ədədlər** çatışmır?"). Alternativləri görmək — düşüncə çevikliyidir.

:::note[bits/stdc++.h]
`#include <bits/stdc++.h>` — bütün standart kitabxanaları bir sətirlə qoşan CP adətidir (GCC-də işləyir). Bundan sonra nümunələrdə bunu işlədəcəyik.
:::

## Çalışmalar

**Çalışma 1.** Ədədləri oxuyun (sayı bilinmir), sonra: neçə ədəd öz **indeksinə** bərabərdir (0-dan)?

<details>
<summary>Həlli</summary>

```cpp
vector<int> a;
int x;
while (cin >> x) a.push_back(x);

int say = 0;
for (int i = 0; i < (int)a.size(); i++)
    if (a[i] == i) say++;
cout << say;
```

</details>

**Çalışma 2.** Vectordan bütün cüt ədədləri silin — `erase`-siz, O(n)-də.

<details>
<summary>Həlli</summary>

```cpp
vector<int> b;
for (int x : a)
    if (x % 2 != 0) b.push_back(x);
a = b;
```

"Yenisinə yığ" üsulu — ortadan silmələrdən həmişə sürətlidir. (İncə yol: `a.erase(remove_if(...))` idiomu — maraqlananlar üçün.)

</details>

**Çalışma 3.** n tələbənin ballarını oxuyun, unikal balları artan sırada çap edin.

<details>
<summary>Həlli</summary>

```cpp
sort(a.begin(), a.end());
for (int i = 0; i < (int)a.size(); i++)
    if (i == 0 or a[i] != a[i - 1])
        cout << a[i] << ' ';
```

Sırala + qonşu müqayisəsi (12-ci dərs). Hazır idiom da var: `a.erase(unique(a.begin(), a.end()), a.end());` — amma yalnız sıralanmış vectorda işləyir!

</details>

## Tez-tez edilən səhvlər

1. `vector<int> v(n)` ilə `v.push_back(...)` qarışdırmaq — nəticə 2n element!
2. `(int)` çevirmədən `size() - 1`.
3. Dövr içində `insert/erase` ilə O(n²).
4. `accumulate`-də `0` (int) başlanğıcı ilə overflow.
5. Boş vectorda `back()` çağırmaq — runtime error.
