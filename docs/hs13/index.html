Həll saatı 12. Sadə sıralama alqoritmləri

Sıralama CP-nin ən fundamental əməliyyatıdır. Praktikada hazır sort() istifadə edəcəyik, amma əvvəlcə sıralamanın necə işlədiyini bilmək lazımdır.

Bu dərsdə öyrənəcəyik:

  • Selection sort.
  • Bubble sort.
  • Insertion sort.

Selection sort​

İdeya: hər addımda qalan hissənin minimumunu tapıb yerinə qoy.

for (int i = 0; i < n - 1; i++) {
    int minYer = i;
    for (int j = i + 1; j < n; j++)
        if (a[j] < a[minYer]) minYer = j;
    swap(a[i], a[minYer]);
}

[7, 2, 9, 4] üzərində izləyək:

AddımMassivNəticə
i=0[2, 7, 9, 4]min=2, 7 ilə dəyişdi
i=1[2, 4, 9, 7]qalanda min=4, 7 ilə dəyişdi
i=2[2, 4, 7, 9]qalanda min=7, 9 ilə dəyişdi

Hər addımdan sonra sol hissə qəti sıralanmışdır. Müqayisə sayı: n(n−1)/2.

Bubble sort (qabarcıq)​

İdeya: qonşu cütləri müqayisə et, səhv sırada olanları dəyiş. Bu zaman böyüklər sağa gedir.

for (int i = 0; i < n - 1; i++) {
    bool ok = false;
    for (int j = 0; j + 1 < n - i; j++) {
        if (a[j] > a[j + 1]) {
            swap(a[j], a[j + 1]);
            ok = true;
        }
    }
    if (!ok) break;   // Heç nə dəyişməsə, demək sıralıdır
}

İki optimallaşdırmaya baxın:

  1. Daxili sərhəd n - i: hər gedişdən sonra ən böyük element artıq sondadır, ona baxmağa ehtiyac deyil.
  2. ok flagı: massiv artıq sıralıdırsa, bir gedişlə bitiririk (ən yaxşı hal O(n)).

Insertion sort (daxiletmə)​

İdeya: Hər yeni elementi solda artıq sıralı hissəyə düzgün yerə yerləşdir.

for (int i = 1; i < n; i++) {
    int x = a[i];
    int j = i - 1;
    while (j >= 0 and a[j] > x) {
        a[j + 1] = a[j];
        j--;
    }
    a[j + 1] = x;
}

Müqayisə​

AlqoritmƏn pisƏn yaxşıNə vaxt düşün
SelectionO(n²)O(n²)ən az swap lazımdırsa
BubbleO(n²)O(n) (bayraqla)demək olar heç vaxt :)
InsertionO(n²)O(n)demək olar sıralı verilənlər

n ≤ 5000 üçün hamısı keçir. n = 10⁵ üçün heç biri keçmir, çünki orada O(n log n) lazımdır.

Niyə əl ilə öyrənirik?

Üç səbəb: (1) bəzi məsələlər sıralamanın prosesini soruşur ("neçə swap oldu?", "k addımdan sonra massiv necədir?"); (2) insertion sort-un "tərs cütlər" əlaqəsi ayrıca məsələ sinfidir; (3) müsahibələrdə və olimpiada nəzəri suallarında birbaşa soruşulur.

Sıralamanın tətbiqləri​

Sıralanmış massivdə çox şey asanlaşır:

// (əvvəlcə sıralayın)
int mn = a[0], mx = a[n - 1];          // min/max: kənarlar
int median = a[n / 2];                  // median: orta

// Fərqli qiymətlərin sayı
int say = 1;
for (int i = 1; i < n; i++)
    if (a[i] != a[i - 1]) say++;

// Ən yaxın cütün fərqi
int minFerq = 2000000000;
for (int i = 1; i < n; i++)
    minFerq = min(minFerq, a[i] - a[i - 1]);

Sonuncuya diqqət: sıralamadan sonra ən yaxın iki ədəd mütləq qonşudur. n² cüt yoxlamaq əvəzinə n−1 qonşuya baxırıq.

Çalışmalar​

Çalışma 1. Selection sort-u elə dəyişin ki, massiv azalan sırada düzülsün.

Həlli
for (int i = 0; i < n - 1; i++) {
    int maksYer = i;
    for (int j = i + 1; j < n; j++)
        if (a[j] > a[maksYer]) maksYer = j;
    swap(a[i], a[maksYer]);
}

Yalnız müqayisə işarəsi dəyişir.

Çalışma 2. Bubble sort neçə swap edir? [5, 1, 4, 2] üçün əl ilə sayın, sonra kodla yoxlayın.

Həlli

Əl ilə: (5,1) dəyiş ➝ [1,5,4,2]; (5,4) ➝ [1,4,5,2]; (5,2) ➝ [1,4,2,5]; (4,2) ➝ [1,2,4,5]. Cəmi 4 swap.

Maraqlı fakt: bubble sort-un swap sayı = massivdəki tərs cütlərin (i < j, a[i] > a[j]) sayı.

Çalışma 3. n tələbənin adı və balı verilir. Onları bala görə azalan sırala (bərabər balda ada görə əlifba sırası). Hələlik iki paralel massivlə həll edin.

Həlli
string ad[1005];
int bal[1005];
// ... oxuma ...

for (int i = 0; i < n - 1; i++) {
    int en = i;
    for (int j = i + 1; j < n; j++) {
        if (bal[j] > bal[en] or (bal[j] == bal[en] and ad[j] < ad[en]))
            en = j;
    }
    swap(bal[i], bal[en]);
    swap(ad[i], ad[en]);   // İKİ massivi birlikdə dəyiş!
}

Tez-tez edilən səhvlər​

  1. Daxili dövrün sərhədini səhv qoyub a[j+1]-də kənara çıxmaq.
  2. Selection sort-da swap-ı daxili dövrün içində etmək (yalnız sonda!).
  3. Paralel massivlərdən yalnız birini dəyişmək.
  4. O(n²) sıralamanı n = 10⁵-ə tətbiq edib TLE almaq.
  5. Insertion sort-da a[j] >= x yazıb bərabər elementlərin sırasını pozmaq.
