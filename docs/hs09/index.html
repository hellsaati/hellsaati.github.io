Həll saatı 8. Simvollar və stringlər

Mətnlə işləmək CP-nin böyük hissəsidir. String simvollar massividir və massivlə oxşar formada işləyir.

Bu dərsdə öyrənəcəyik:

  • char tipi və ASCII kodları
  • stringlər üzərində iş
  • Hərf/rəqəm çevrilmələri və ən çox işlənən string əməliyyatları

char və ASCII​

Hər simvol yaddaşda kiçik tam ədəd ASCII kodu kimi saxlanır: 'A' = 65, 'a' = 97, '0' = 48. Aşağıdakı nümunələrə baxaq:

char c = 'f';

            // 1. Hərfin əlifbada yeri
            
int yer = c - 'a';              // 5 (a=0, b=1, ...)

            // 2. Böyükdən kiçiyə / kiçikdən böyüyə

char big = c - 'a' + 'A';       // 'F'
char small = 'D' - 'A' + 'a';   // 'd'

            // 3. Rəqəm simvolu ➝ ədəd

int r = '7' - '0';              // 7 (int olaraq)

Aşağıdakı hazır funksiyalardan da istifadə edə bilərisiz (#include <cctype>): isdigit(c), isalpha(c), islower(c), isupper(c), tolower(c), toupper(c).

string tipi​

#include <string>

string str = "salam";
cout << str.size();      // 5: uzunluq
cout << str[0];          // 's': massiv kimi indeksləmə
str[0] = 'S';            // dəyişmək olar
str += " dünya";         // s = "Salam dünya" birləşdirmə
s.size() və mənfi ədədlər

s.size() işarəsiz (unsigned) tip qaytarır. s.size() - 1 boş stringdə nəhəng müsbət ədədə çevrilir: i < s.size() - 1 kimi şərtlər problem yarada bilər. Ona görə də (int)s.size() yazın və ya uzunluğu int len = s.size(); kimi saxlayın.

Oxuma​

string s;
cin >> s;              // bir söz daxil edir
getline(cin, s);       // bütün sətri daxil edir

Əgər cin >> n-dən sonra getline işlətsəniz, buferdə qalan \n səbəbindən boş string oxunar. Ona görə də aralarında cin.ignore(); istifadə edin.

Gəzmə​

// Klassik
for (int i = 0; i < (int)s.size(); i++)
    if (s[i] == 'a') say++;

// Range-based
for (char c : s)
    if (c == 'a') say++;

Müqayisə və dilimləri​

if (s == t) ...                 // bərabərlik
if (s < t) ...                  // leksikoqrafik sıra (lüğət sırası)

string parca = s.substr(2, 3);  // 2-ci indeksdən 3 simvol
int yer = s.find("ab");         // ilk mövqe, tapılmasa string::npos
if (s.find("ab") != string::npos) cout << "var";

Klassik Templatelər​

Hərflərin tezliyi​

int say[26] = {0};
for (char c : s)
    say[c - 'a']++;

Palindrom​

bool palindrome(string &s) {
    int l = 0, r = (int)s.size() - 1;
    while (l < r) {
        if (s[l] != s[r]) return 0;
        l++; r--;
    }
    return 1;
}

Registrsiz müqayisə (Petya and Strings)​

string a, b;
cin >> a >> b;
for (char &c : a) c = tolower(c);
for (char &c : b) c = tolower(c);

if (a < b) cout << -1;
else if (a > b) cout << 1;
else cout << 0;

Söz saymaq​

string str;
int cv = 0;
while (cin >> str) cv++;

cin >> str oxuma uğursuz olanda (giriş bitəndə) yanlış qaytarır və dövr avtomatik dayanır.

Nümunə: CSES Repetitions​

DNA sətri verilir (yalnız A, C, G, T). Ən uzun eyni simvollu ardıcıl parçanın uzunluğunu tapın.

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;

    int le = 1, cv = 1;
    for (int i = 1; i < (int)s.size(); i++) {
        if (s[i] == s[i - 1]) le++;
        else le = 1;
        cv = max(cv, le);
    }
    cout << cv;
    return 0;
}

Çalışmalar​

Çalışma 1. Verilmiş stringdə sait hərflərin (a, e, i, o, u) sayını tapın.

Həlli
int cv = 0;
for (char c : s) {
    c = tolower(c);
    if (c == 'a' or c == 'e' or c == 'i' or c == 'o' or c == 'u')
        cv++;
}

Alternativ: string ls = "aeiou"; if (ls.find(c) != string::npos) cv++;

Çalışma 2. Hazır funksiya işlətmədən sətri tərsinə çevirin.

Həlli
for (int i = 0; i < (int)s.size() / 2; i++)
    swap(s[i], s[s.size() - 1 - i]);

Massivi tərsinə çevirmə ilə eynidir. (Hazır yol: reverse(s.begin(), s.end());)

Çalışma 3. Verilmiş stringdə hər söz böyük hərflə başlasın ("salam necəsən" ➝ "Salam Necəsən").

Həlli
string s;
getline(cin, s);

for (int i = 0; i < (int)s.size(); i++) {
    if (i == 0 or s[i - 1] == ' ')
        s[i] = toupper(s[i]);
}
cout << s;

"Sözün əvvəli" = ya sətrin əvvəli, ya boşluqdan sonrakı simvol.

Tez-tez edilən səhvlər​

  1. s.size() - 1 unsigned problemi;
  2. cin >> və getline qarışığında \n buferi;
  3. '7' (simvol) ilə 7 (ədəd) qarışdırmaq;
  4. == ilə C-based stringləri (char massivi) müqayisə etməyə çalışmaq;
  5. Range-based for-da & unutmaqla orijinalı dəyişməmək.
