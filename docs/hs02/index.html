Həll saatı 1. Şərt operatorları

Proqramlar yalnız yuxarıdan aşağı sətir-sətir işləsəydi, kompüter sadə kalkulyatordan fərqlənməzdi. Proqramı "ağıllı" edən əsas alət şərt operatorlarıdır. Şərt operatorları müəyyən şərt ödəndikdə bir işi, ödənmədikdə başqa işi görməyə kömək edir.

Bu dərsdə öyrənəcəyik:

  • if, else, else if konstruksiyaları
  • Müqayisə operatorları və məntiqi operatorlar (&&, and, ||, or, !, not)
  • Şərtlərin iç-içə yazılması və tez-tez edilən səhvlər
Vaxt planı

Bu dərsdə nəzəri olaraq çox bilik yoxdur. Kodları mütləq özünüz yazıb işlədin, sadəcə üzdən oxumaq bəs deyil.

Müqayisə operatorları​

C++-da iki qiyməti müqayisə etmək üçün bu operatorlardan istifadə olunur:

OperatorMənasıNümunə (a = 5, b = 3)
==bərabərdira == b: false
!=bərabər deyila != b: true
>böyükdüra > b: true
<kiçikdira < b: false
>=böyük və ya bərabəra >= 5: true
<=kiçik və ya bərabərb <= 2: false

Müqayisənin nəticəsi bool tipindədir: true (doğru) və ya false (yanlış). C++ daxildə true-nu 1, false-u 0 kimi saxlayır.

Ən klassik səhv

= mənimsətmə, == müqayisədir. if (x = 5) yazsanız, kod xəta vermir: x-ə 5 mənimsədilir və şərt həmişə doğru olur.

if operatoru​

Ən sadə forma:

#include <iostream>
using namespace std;

int main() {
    int yas;
    cin >> yas;

    if (yas >= 18) {
        cout << "Sürücülük kursuna müraciət edə bilərsiniz!";
    }
    return 0;
}

if-dən sonrakı mötərizədəki şərt doğrudursa, fiqurlu mötərizələr arasındakı blok işləyir; yanlışdırsa, blok ötürülür.

Şərt yalnız bir sətirdirsə, fiqurlu mötərizələri yazmamaq olar. Amma yeni başlamısınızsa, bu mötərizələri yazmaq kodu daha anlaşıqlı etdiyindən yazmağınız daha yaxşı olar:

if (yas >= 18)
    cout << "Salam";    // yalnız bu sətir if-ə aiddir
    cout << "Sağol";    // bu sətir isə həmişə işləyir

İkinci cout görünüşcə if-in içindədir, amma deyil. Fiqurlu mötərizə olmadıqda if yalnız bir növbəti ifadəni idarə edir.

else və else if​

int n;
cin >> n;

if (n > 0) {
    cout << "Müsbət";
} else if (n < 0) {
    cout << "Mənfi";
} else {
    cout << "Sıfır";
}

Zəncir yuxarıdan aşağı yoxlanılır və ilk doğru şərtin bloku işləyir, qalanları ötürülür. Ona görə şərtlərin sırası vacibdir:

int bal;
cin >> bal;

if (bal >= 90) cout << "A";
else if (bal >= 80) cout << "B";
else if (bal >= 70) cout << "C";
else cout << "F";

Burada bal = 95 üçün yalnız A çap olunur, baxmayaraq ki, 95 >= 80 də doğrudur. Əgər else-siz, ayrı-ayrı if-lər yazsaydıq, A, B və C üçü də çap olunardı. Fərqi dərk etmək üçün hər iki variantı işlədib müqayisə edin.

Məntiqi operatorlar​

Bir neçə şərti birləşdirmək üçün üç operator var:

  • &&, and (VƏ): hər iki tərəf doğru olduqda doğrudur
  • ||, or (VƏ YA): ən azı bir tərəf doğru olduqda doğrudur
  • !, not (İNKAR): doğrunu yanlışa, yanlışı doğruya çevirir
int yas, bal;
cin >> yas >> bal;

if (yas >= 14 and yas <= 18 and bal >= 50) {
    cout << "Kriteriyalara uyğundur.";
}

if (bal < 50 or yas > 18) {
    cout << "Kriteriyalara uyğun deyil.";
}

Riyaziyyatda yazdığımız 14 <= yas <= 18 formasını C++-da yazmaq olmaz:

if (14 <= yas <= 18)            // Bu ifadə həmişə `true` qaytarır
if (14 <= yas and yas <= 18)    // Doğru yazılış forması

Səbəb: 14 <= yas əvvəl hesablanır, nəticə 0 və ya 1 olur, sonra 0 <= 18 və ya 1 <= 18 yoxlanılır, bu isə həmişə doğrudur.

Qısa qapanma (short-circuit)

and-də sol tərəf yanlışdırsa, sağ tərəf heç hesablanmır. or-də sol tərəf doğrudursa, sağ tərəf hesablanmır. Bu, gələcəkdə if (i < n and a[i] == x) kimi yoxlamalarda çox işinizə yarayacaq.

Yazılış qaydası

Əgər olimpiada ilə, vəya ümumiyyətlə proqramlaşdırma ilə yeni məşğul olmağa başlamısınızsa, çalışın &&, ||, ! yerinə and, or və not işlədin. Bu, kodunuzu daha rahat başa düşməyinizə kömək edəcək.

Cüt, yoxsa tək?​

Olimpiada məsələlərində ən çox rast gəlinən şərt, qalığın yoxlamasıdır. % operatoru bölmədən qalığı verir:

int n;
cin >> n;

if (n % 2 == 0) cout << n << " cütdür";
else cout << n << " təkdir";

Eyni məntiqlə: n % 10 10-a qalıq, n % 5 == 0 5-ə bölünür, və s.

İç-içə şərtlər​

if-in içində yeni if yazmaq olar:

int a, b, c;
cin >> a >> b >> c;

if (a >= b) {
    if (a >= c) cout << a;
    else cout << c;
} else {
    if (b >= c) cout << b;
    else cout << c;
}

Bu kod üç ədədin maksimumunu tapır. Eyni məsələni məntiqi operatorlarla da yazmaq olar:

if (a >= b and a >= c) cout << a;
else if (b >= a and b >= c) cout << b;
else cout << c;

Hər iki yanaşma düzgündür. Kod nə qədər az iç-içə olsa, oxumaq bir o qədər asandır.

Üçlük operator (ternary)​

Qısa şərtlər üçün C++-da bir sətirlik forma var:

int maks = (a > b) ? a : b;
// əgər a > b doğrudursa maks = a, deyilsə maks = b
Qeyd

Bunu bilmək faydalıdır, amma mürəkkəb şərtlərdə istifadə etməyin. Ternary operatorlar uzun şərtlərdə oxunaqlı olmur.

Nümunə​

Codeforces 4A ("Watermelon"): w kiloqramlıq qarpızı iki hissəyə bölmək lazımdır ki, hər hissə cüt və müsbət çəkidə olsun.

Fikirləşək: iki cüt ədədin cəmi cütdür. Deməli w cüt olmalıdır. Amma bir istisna var: w = 2 olduqda hissələr 1+1-dən başqa cür alınmır, halbuki 1 tək ədəddir. Deməli cavab:

#include <iostream>
using namespace std;

int main() {
    int w;
    cin >> w;

    if (w % 2 == 0 and w > 2) cout << "YES";
    else cout << "NO";
    return 0;
}
Dərsin ən vacib fikri

w = 2 kimi hallara kənar hal (edge case) deyilir. Şərt qurarkən həmişə soruşun: "ən kiçik / ən böyük / sərhəddəki qiymətlərdə kodum nə edəcək?" Bu vərdişi 13-cü həll saatında ayrıca dərinləşdirəcəyik.

Çalışmalar​

Hər çalışmanı əvvəlcə özünüz yazın, sonra həlli açın.

Çalışma 1. Verilmiş n ilinin uzun il (366 gün) olub-olmadığını tapın. Qayda: il 4-ə bölünürsə uzundur, amma 100-ə bölünürsə uzun deyil, amma 400-ə bölünürsə yenə uzundur.

Həlli
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    if ((n % 4 == 0 and n % 100 != 0) or n % 400 == 0)
        cout << "YES";
    else
        cout << "NO";
    return 0;
}

Mötərizələrə diqqət: and operatoru or operatorundan daha üstündür, amma mötərizə qoymaq həm oxunaqlıdır, həm də belə gözdən qaça biləcək səhvlərdən qoruyur.

Sual 2. Üç tərəfin uzunluğu verilir. Bu tərəflərdən üçbucaq qurmaq mümkündürmü? (Üçbucaq bərabərsizliyi: istənilən iki tərəfin cəmi üçüncüdən böyük olmalıdır.)

Həlli
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;

    if (a + b > c and a + c > b and b + c > a)
        cout << "YES";
    else
        cout << "NO";
    return 0;
}

Üç şərtin hamısı yoxlanmalıdır. Əgər yalnız birini yoxlamısınızsa, həlliniz səhv olaraq qəbul olunur.

Sual 3. Verilmiş 3 rəqəmli ədədin rəqəmləri cəmini tapın (məsələn, 345: 12).

Həlli
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int r1 = n % 10;        // sonuncu rəqəm
    int r2 = (n / 10) % 10; // ortadakı rəqəm
    int r3 = n / 100;       // birinci rəqəm
    cout << r1 + r2 + r3;
    return 0;
}

Tez-tez edilən səhvlər​

  1. = və == qarışdırmaq;
  2. 14 <= x <= 18 kimi yazmaq;
  3. Fiqurlu mötərizəni buraxıb iki sətri if-ə aid etmək;
  4. else if zəncirində şərtlərin sırasını səhv qurmaq;
  5. Edge case-ləri (ən kiçik qiymət, bərabərlik halı) unutmaq.
