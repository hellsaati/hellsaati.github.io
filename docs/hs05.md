# Həll saatı 4. Massivlər: əsaslar

İndiyə qədər ədədləri oxuyub dərhal emal edirdik. Amma çox vaxt verilənləri yadda saxlamaq lazım olur: "ədədləri oxu, sonra axırdan əvvələ çap et" kimi sualları massivsiz etmək olmur.

**Massiv** eyni tipli elementlərin nömrələnmiş siyahısıdır.

Bu dərsdə öyrənəcəyik:

- Massivin yazılışı, doldurulması, çapı
- 0-dan indeksləmə və sərhəd xətaları
- Massivlə tipik ilk məsələlər

## Elan və indekslər

```cpp
int a[5];       // 5 elementlik massiv (içindəki qiymətlər random ədədlərdən ibarətdir)
int b[5] = {7, 2, 9, 4, 1};

cout << b[0];   // 7, birinci element 0 nömrəlidir
cout << b[4];   // 1, sonuncu element n-1 nömrəlidir
```

Ən vacib qayda: n elementlik massivin indeksləri 0-dan n−1-ə qədərdir. `b[5]`-ə müraciət massivdən kənara çıxmaqdır.

:::danger[Sərhəddən kənara çıxma]
`b[5]`, `b[-1]` kimi müraciətlər C++-da xəta mesajı vermir, əksinə, proqram yaddaşın "qonşu" hissəsini idarə edir. Nəticə gah düz cavab, gah zibil, gah da çökmə (runtime error) ola bilər. Bu, C++-da ən gözdən qaça bilən səhvdir. İndeksin sərhəddə olduğuna həmişə əmin olun.
:::

Massivin ölçüsü kompilyasiya zamanı sabit olmalıdır. CP-də adət olaraq maksimal ölçünü yuxarıda sabitlə elan edirik:

```cpp
const int maxn = 100005;
int a[maxn];

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    ...
}
```

Qlobal (main-dən kənarda) declare olunan massiv avtomatik sıfırlarla dolur və çox böyük ola bilər (10⁷ element). Lakin, lokal massivlər üçün limit xeyli azdır (≈10⁵–10⁶). Buna görə də, massivləri qlobalda declare etmək çox zaman daha məntiqlidir.

## Doldurma və çap

```cpp
int n;
cin >> n;
for (int i = 0; i < n; i++) cin >> a[i];

for (int i = 0; i < n; i++) cout << a[i] << ' ';
cout << '\n';

for (int i = n - 1; i >= 0; i--) cout << a[i] << ' ';  // tərsinə
```

Keçən dərsin template-ləri (cəm, maksimum, sayğac) massivlə eyni cür işləyir, üstəlik indi verilənlərə bir neçə dəfə baxa bilərik:

```cpp
// Maksimum element massivdə neçə dəfə rast gəlinir?
int maks = a[0];
for (int i = 1; i < n; i++)
    if (a[i] > maks) maks = a[i];

int say = 0;
for (int i = 0; i < n; i++)
    if (a[i] == maks) say++;
```
:::tip
İlk elementi başlanğıc götürmək (`maks = a[0]`) mənfi ədədlər olduqda da işləyən ən etibarlı üsuldur.
:::

## Sayma massivi (ilk "data strukturumuz")

Qiymətlər kiçik diapazondadırsa (məsələn, 1..100), hər qiymətin sayını ayrıca massivdə saxlaya bilərik:

```cpp
int say[101] = {0};   // say[v] = v qiyməti neçə dəfə gəlib

for (int i = 0; i < n; i++) {
    int x;
    cin >> x;
    say[x]++;
}

// Ən çox rast gəlinən qiymət:
int ans = 1;
for (int v = 1; v <= 100; v++)
    if (say[v] > say[ans]) ans = v;
```

Bu sadə "counting" ideyası gələcəkdə dəfələrlə qarşınıza çıxacaq: sayma sıralaması, hash cədvəllər, tezlik analizləri və s. hamısı buradan başlayır.

## Nümunə: Beautiful Matrix üslubunda

5 x 5 cədvəldə tək 1 var, onu mərkəzə (3, 3) gətirmək üçün neçə addım lazımdır?

```cpp
#include <iostream>
using namespace std;

int main() {
    int setir = 0, sutun = 0;
    for (int i = 1; i <= 5; i++) {
        for (int j = 1; j <= 5; j++) {
            int x;
            cin >> x;
            if (x == 1) { setir = i; sutun = j; }
        }
    }
    int ferq1 = setir - 3;
    if (ferq1 < 0) ferq1 = -ferq1;
    int ferq2 = sutun - 3;
    if (ferq2 < 0) ferq2 = -ferq2;
    cout << ferq1 + ferq2;
    return 0;
}
```

Mütləq qiymət üçün hazır funksiya da var: `abs(x)`.

## Çalışmalar

**Çalışma 1.** n ədəd oxuyun, sonra hamısını əks sırada çap edin.

<details>
<summary>Həlli</summary>

```cpp
#include <iostream>
using namespace std;

int a[100005];

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = n - 1; i >= 0; i--) cout << a[i] << ' ';
    return 0;
}
```

</details>

**Çalışma 2.** n ədəd verilir (1 ≤ n ≤ 100000). Ən böyük elementin indeksini (1-dən nömrələməklə) tapın. Əgər bir neçə dənə varsa, ilk gələnin indeksini tapın.

<details>
<summary>Həlli</summary>

```cpp
#include <iostream>
using namespace std;

int a[100005];

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];

    int yer = 0;
    for (int i = 1; i < n; i++)
        if (a[i] > a[yer]) yer = i;   // `>` ilk maksimumu saxlayır

    cout << yer + 1;   // 1-dən nömrələmə
    return 0;
}
```

:::note[`>` və `>=` fərqi]
Diqqət: `>` yazsaq ilk maksimum, `>=` yazsaq son maksimum tapılır.
:::

</details>

**Çalışma 3.** n ədəd verilir (1 ≤ ədədlər ≤ 1000). Neçə fərqli qiymət var?

<details>
<summary>Həlli</summary>

```cpp
#include <iostream>
using namespace std;

bool var[1001];

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        var[x] = true;
    }
    int say = 0;
    for (int v = 1; v <= 1000; v++)
        if (var[v]) say++;
    cout << say;
    return 0;
}
```

Sayma massivinin `bool` variantı. Əgər diapazon böyük olsaydı (10⁹-a qədər), bu üsul işləməzdi. Bu halda nə edəcəyimizi gələn dərsdə öyrənəcəyik.

</details>

## Tez-tez edilən səhvlər

1. `a[n]`-ə müraciət;
2. Böyük massivi `main` daxilində elan edib runtime error almaq;
3. Lokal massivin ilkin qiymətlərinin zibil olduğunu unutmaq;
4. Massivin ölçüsü olaraq dəyişən götürmək(bəzi kompilyatorlarda işləsə də, standart deyil. `const int maxn` kimi bir dəyişən işlədin);
5. 1-dən nömrələnən cavab istəyəndə 0-dan nömrələnmiş indeksi çap etmək.
