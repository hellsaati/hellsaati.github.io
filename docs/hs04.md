# Həll saatı 3. İç-içə dövrlər

Dövrün içində dövr yazmaq CP-də ən çox istifadə olunan konstruksiyalardan biridir. Matrislər, fiqur çapı, bütün cütlərin yoxlanması, sadə ədəd testi və s. mövzularda iç-içə dövrlərə tez tez ehtiyac duyulur. Bu dərsdə:

- İç-içə dövrün iş prinsipi və addım sayı
- Fiqur/naxış çapı məsələləri
- Bütün cütləri gəzmək və sadəlik yoxlaması
- İç-içə dövrdən düzgün çıxmaq (`break` incəlikləri)

## İş prinsipi

```cpp
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 4; j++) {
        cout << i << j << ' ';
    }
    cout << '\n';
}
```

Nəticə:

```
11 12 13 14
21 22 23 24
31 32 33 34
```

Qayda belədir: Xarici dövrün hər addımında daxili dövr tam işləyir. `i = 1` olarkən `j` 1-dən 4-ə qədər gedir, sonra `i = 2` olur və `j` yenidən 1-dən başlayır.

Ümumi addım sayı: 3 * 4 = 12. Ümumiyyətlə, `n` addımlı dövrün içində `m` addımlı dövr `n * m` addım deməkdir.

:::tip[Sürət təxmini]
Kompüter saniyədə təxminən 10⁸–10⁹ sadə əməliyyat yerinə yetirir. `n = 1000` üçün iç-içə dövr 10⁶ addımdır, yəni ani işləyir. `n = 100000` üçün isə 10¹⁰ addım dəqiqələrlə çəkər və TLE alarsınız. Məsələ həll etməzdən əvvəl həmişə addım sayını qiymətləndirin.
:::

## Fiqur çapı

Klassik başlanğıc məsələləri. `n = 4` üçün:

Kvadrat şəkli
```cpp
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n; j++)
    cout << "*";
}
```
```
****
****
****
****
```

Üçbucaq şəkli
```cpp
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++)
    cout << "*";
    cout << '\n';
}
```

```
*
**
***
****
```

Üçbucaqda sirr daxili dövrün sərhədindədir: `j <= i`. Yəni, `i`-ci sətirdə `i` ulduz olmalıdır. Əgər sərhədi `j <= n - i + 1` etsək, tərs üçbucaq alınar.

**Çalışma (özünüz həll edin):** ortalanmış piramida çap edin: `i`-ci sətirdə əvvəl `n - i` boşluq, sonra `2 * i - 1` ulduz olsun.

## Bütün cütləri gəzmək

"n ədəd arasında fərqi `k` olan neçə cüt var?" tipli məsələlərdə bütün `(i, j)` cütlərini yoxlayırıq:

```cpp
int say = 0;
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        if (a[j] - a[i] == k or a[i] - a[j] == k) say++;
    }
}
```

Daxili dövrün `j = i + 1`-dən başlaması iki işi birdən görür: Cüt öz-özü ilə müqayisə olunmur və hər cüt yalnız bir dəfə sayılır. Addım sayı: n * (n−1)/2 ≈ n²/2.

## İç-içə dövrdə break

`break` yalnız **öz** dövrünü dayandırır:

```cpp
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        if (a[i] + a[j] == x) break;  // yalnız daxili dövrü dayandırır
    }
}
```

Hər iki dövrdən çıxmağın iki təmiz yolu:

```cpp
// 1. "flag" işlətmək
bool tapildi = false;
for (int i = 0; i < n and !tapildi; i++) {
    for (int j = 0; j < n; j++) {
        if (a[i] + a[j] == x) { tapildi = true; break; }
    }
}

// 2. Funksiyada return etmək (gələn dərslərdə keçəcəyik)
```

## Nümunə: vuruqlara ayırma

`n` verilir, onu sadə vuruqlara ayırın (məsələn, `60: 2 2 3 5`):

```cpp
#include <iostream>
using namespace std;

int main() {
    long long n;
    cin >> n;

    for (long long d = 2; d * d <= n; d++) {
        while (n % d == 0) {   // for-un içində while
            cout << d << ' ';
            n /= d;
        }
    }
    if (n > 1) cout << n;      // qalan hissə sadə ədəddir
    return 0;
}
```

Diqqət: daxili dövr `while`-dır, eyni vuruq bir neçə dəfə çıxa bilər. Sondakı `if (n > 1)` edge case-ni unutmayın: Əgər ədədin tərkibində ədədin kökaltından böyük bir sadə vuruq varsa, sonda ona da baxmalıyıq.

## Çalışmalar

**Çalışma 1.** `n * n` vurma cədvəlini çap edin.

<details>
<summary>Həlli</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++)
            cout << i * j << ' ';
        cout << '\n';
    }
    return 0;
}
```

</details>

**Çalışma 2.** `n` ədəd verilir. Aralarında bərabər olan cüt varmı?

<details>
<summary>Həlli</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, a[1000];
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];

    bool ok = false;
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            if (a[i] == a[j]) ok = true;

    cout << (ok ? "YES" : "NO");
    return 0;
}
```

n² üsul kiçik n üçün kifayətdir. Gələn dərslərdə bu həlli necə sürətləndirə biləcəyimiz haqqında daha ətraflı danışacayıq.

</details>

**Çalışma 3.** 2-dən n-ə qədər hər ədədin ən kiçik sadə bölənini çap edin.

<details>
<summary>Həlli</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    for (int i = 2; i <= n; i++) {
        for (int d = 2; d <= i; d++) {
            if (i % d == 0) {
                cout << i << ": " << d << '\n';
                break;   // ilk böləni tapan kimi dövrü dayandır
            }
        }
    }
    return 0;
}
```

Burada `break` ən kiçik bölən tapılan kimi daxili dövrü dayandırır.

</details>

## Tez-tez edilən səhvlər

1. Daxili dövrdə xarici dövrün sayğacını (`i`) artırmaq/dəyişmək;
2. Hər iki dövrdə eyni sayğac adı (`i`) istifadə etmək;
3. `break`-in yalnız daxili dövrü dayandırdığını unutmaq;
4. Addım sayını qiymətləndirməyib n² həlli böyük n-ə tətbiq etmək;
5. Sətir sonu `'\n'`-i daxili dövrün içinə yazmaq.
